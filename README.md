# 🎧 Spotify Churn Analysis

## 📌 Overview
This project analyzes user behavior data from a music streaming platform to understand customer churn patterns. The goal is to identify key behavioral signals that influence churn and provide actionable insights to improve user retention.

---

## 🎯 Objective
- Identify factors influencing user churn behavior  
- Analyze engagement patterns of active vs churned users  
- Understand relationship between listening behavior and retention  
- Segment users based on engagement levels  

---

## 🛠 Tools & Technologies
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Google Colab Notebook  
- Exploratory Data Analysis (EDA)

---

## 📊 Key Insights
- Low-engagement users are the highest churn-risk group  
- Listening time, engagement score, and songs played are highly correlated metrics  
- Churned users show significantly lower activity levels compared to retained users  
- Mid-engagement users show unstable behavior patterns between churned and active segments  
- Free users have higher churn compared to premium users  
- Mobile and web users show similar churn behavior patterns  

---

## 🔍 Methodology
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Correlation analysis between engagement metrics  
- User segmentation based on activity levels  
- Visualization of churn behavior patterns  
- Insight extraction for business understanding  

---

## ❓ Problem Statement
Music streaming platforms often struggle with user churn, where users stop engaging with the service over time. The challenge is to identify behavioral patterns that differentiate churned users from retained users and understand which engagement signals can help predict churn early.

---

## 📊 Key Findings
- Low-engagement users are the highest churn-risk group  
- Listening time, engagement score, and songs played are strongly correlated and represent similar user behavior patterns  
- Churned users consistently show lower activity levels compared to retained users  
- Mid-engagement users exhibit unstable behavior patterns and may transition to either churn or retention  
- Free users have significantly higher churn compared to premium users  
- Mobile and web users show similar churn behavior patterns  
- Younger users show slightly higher variability in engagement levels  

---

## 💡 Proposed Solution
- Identify at-risk users early using signals such as low engagement score, reduced listening time, and low daily activity  
- Focus retention strategies on low and mid-engagement user segments  
- Improve engagement of free users through personalized recommendations and targeted features  
- Monitor correlated metrics (listening time, engagement score, songs played) as a combined engagement indicator  
- Use segmentation-based insights to design targeted churn prevention strategies

 ---

## 🧾 Conclusion
This project demonstrates how user behavior data can be transformed into actionable insights for churn analysis. By identifying high-risk user segments and understanding engagement patterns, businesses can design targeted retention strategies and improve overall user lifetime value.
